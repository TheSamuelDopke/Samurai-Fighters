## 📖 Sobre  
Samurai Fighters é um jogo de luta 2D desenvolvido com HTML5 Canvas e JavaScript. O jogador controla um samurai e deve derrotar inimigos em batalhas épicas.  

## 🎮 Controles  
- **Setas/WASD** - Movimentação  
- **Barra de espaço** - Ataque
